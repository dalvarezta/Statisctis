Understanding Confidence Intervals

A frequentist approach to parameter estimation

What is a Confidence Interval?

A confidence interval (CI) is a range of values, derived from sample data, that is likely to contain the true value of an unknown population parameter. Unlike a point estimate (which returns a single number), a CI quantifies the uncertainty inherent in sampling.

📏

Formal Definition

A \(100(1 - \alpha)\%\) confidence interval for a parameter \(\theta\) is an interval \((L,\, U)\) computed from sample data such that, before sampling:

P(L \leq \theta \leq U) \;=\; 1 - \alpha

Common CI Formulas

1. CI for Population Mean (σ known)

\bar{x} \;\pm\; z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}

Where \(\bar{x}\) is the sample mean, \(z_{\alpha/2}\) is the standard normal critical value, \(\sigma\) is the population standard deviation, and \(n\) is sample size.

2. CI for Population Mean (σ unknown)

\bar{x} \;\pm\; t_{\alpha/2,\, n-1} \cdot \frac{s}{\sqrt{n}}

Uses the Student's \(t\)-distribution with \(n - 1\) degrees of freedom and the sample standard deviation \(s\).

3. CI for Population Proportion

\hat{p} \;\pm\; z_{\alpha/2} \cdot \sqrt{\frac{\hat{p}(1 - \hat{p})}{n}}

Requires \(n\hat{p} \geq 10\) and \(n(1 - \hat{p}) \geq 10\) for the normal approximation to hold.

Interpreting "Confidence"

The confidence level describes the long-run performance of the procedure. A 95% CI method, applied repeatedly across infinite samples, will capture the true parameter 95% of the time. It is the method, not the specific interval, that has a probability attached.

⚠️

Misconception Warning

❌ Wrong Interpretation

"There is a 95% probability that the true mean lies within this specific interval."

✅ Correct Interpretation

"If we repeated this sampling process infinitely, 95% of the intervals we compute would contain the true mean."

In frequentist statistics, the true parameter is a fixed, unknown constant — not a random variable. The interval is what varies from sample to sample.

Once an interval is calculated, it either contains the parameter or it doesn't (probability is exactly 0 or 1). The "95%" refers to the long-run success rate of the procedure, not any single interval.

🔑 Factors Affecting Width

  • ↗ Higher confidence → Wider interval
  • ↘ Larger sample size → Narrower interval
  • ↗ Greater variability (σ) → Wider interval
