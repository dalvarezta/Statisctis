Module 5 · Theory

Understanding Confidence Intervals

A frequentist approach to parameter estimation

What is a Confidence Interval?

A confidence interval (CI) is a range of values, derived from sample data, that is likely to contain the true value of an unknown population parameter. Unlike a point estimate (which returns a single number), a CI quantifies the uncertainty inherent in sampling.

📏

Formal Definition

A \(100(1 - \alpha)\%\) confidence interval for a parameter \(\theta\) is an interval \((L,\, U)\) computed from sample data such that, before sampling:

P(L \leq \theta \leq U) \;=\; 1 - \alpha

Common CI Formulas

1. CI for Population Mean (σ known)

\bar{x} \;\pm\; z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}

Where \(\bar{x}\) is the sample mean, \(z_{\alpha/2}\) is the standard normal critical value, \(\sigma\) is the population standard deviation, and \(n\) is sample size.

2. CI for Population Mean (σ unknown)

\bar{x} \;\pm\; t_{\alpha/2,\, n-1} \cdot \frac{s}{\sqrt{n}}

Uses the Student's \(t\)-distribution with \(n - 1\) degrees of freedom and the sample standard deviation \(s\).

3. CI for Population Proportion

\hat{p} \;\pm\; z_{\alpha/2} \cdot \sqrt{\frac{\hat{p}(1 - \hat{p})}{n}}

Requires \(n\hat{p} \geq 10\) and \(n(1 - \hat{p}) \geq 10\) for the normal approximation to hold.

Interpreting "Confidence"

The confidence level describes the long-run performance of the procedure. A 95% CI method, applied repeatedly across infinite samples, will capture the true parameter 95% of the time. It is the method, not the specific interval, that has a probability attached.

⚠️

Misconception Warning

❌ Wrong Interpretation

"There is a 95% probability that the true mean lies within this specific interval."

✅ Correct Interpretation

"If we repeated this sampling process infinitely, 95% of the intervals we compute would contain the true mean."

In frequentist statistics, the true parameter is a fixed, unknown constant — not a random variable. The interval is what varies from sample to sample.

Once an interval is calculated, it either contains the parameter or it doesn't (probability is exactly 0 or 1). The "95%" refers to the long-run success rate of the procedure, not any single interval.

🔑 Factors Affecting Width

  • ↗ Higher confidence → Wider interval
  • ↘ Larger sample size → Narrower interval
  • ↗ Greater variability (σ) → Wider interval

Module 6 · Theory

Introduction to Hypothesis Testing

A systematic approach to making decisions using sample data

What is Hypothesis Testing?

Hypothesis testing is a statistical method used to make decisions about population parameters based on sample data. It provides a structured framework for evaluating claims or assumptions about a population.

🎯

The Core Idea

We start with an assumption (the null hypothesis), collect data, and determine whether the data provides sufficient evidence to reject that assumption in favor of an alternative explanation.

The Null and Alternative Hypotheses

$H_0$ Null Hypothesis
  • Represents the status quo or no effect
  • Always contains an equality (=, ≤, ≥)
  • Assumed true until evidence suggests otherwise
  • What we test against
Example: $H_0: \mu = 100$ (The population mean equals 100)
$H_1$ or $H_a$ Alternative Hypothesis
  • Represents the research claim or effect exists
  • Contains inequality (≠, <, >)
  • What we conclude if we reject $H_0$
  • Requires sufficient evidence to support
Example: $H_1: \mu \neq 100$ (The population mean differs from 100)

Significance Level ($\alpha$)

The significance level, denoted by $\alpha$ (alpha), is the probability threshold for rejecting the null hypothesis. It represents the risk we're willing to take of making a Type I error.

Common Significance Levels

$\alpha$Confidence LevelInterpretation
0.1090%Less stringent; more likely to reject $H_0$
0.0595%Most commonly used standard
0.0199%Very stringent; strong evidence required

Decision Rule: If the p-value < $\alpha$, reject $H_0$. Otherwise, fail to reject $H_0$.

The P-Value

The p-value is the probability of obtaining test results at least as extreme as the observed results, assuming the null hypothesis is true.

📊

Understanding the P-Value

  • Small p-value (< $\alpha$): The observed data is unlikely under $H_0$ → Reject $H_0$
  • Large p-value (≥ $\alpha$): The observed data is consistent with $H_0$ → Fail to reject $H_0$

Remember: The p-value is NOT the probability that $H_0$ is true!

⚖️ Types of Tests

  • ↔ Two-tailed: $H_1: \mu \neq \mu_0$ (tests for any difference)
  • ← Left-tailed: $H_1: \mu < \mu_0$ (tests if smaller)
  • → Right-tailed: $H_1: \mu > \mu_0$ (tests if larger)

📋 Z-Test vs T-Test

TestWhen to Use
Z-testσ known OR n ≥ 30
T-testσ unknown AND n < 30
⚠️

Important Note

❌ "Accept $H_0$"

We never "accept" the null hypothesis.

✅ "Fail to Reject $H_0$"

We either reject $H_0$ or fail to reject it. Failing to reject doesn't prove $H_0$ is true—it means we lack sufficient evidence against it.

📝 Step-by-Step Hypothesis Testing Workflow

1

State the Hypotheses

Clearly define $H_0$ and $H_1$ based on the research question. Determine if the test is one-tailed or two-tailed.

2

Choose Significance Level ($\alpha$)

Select $\alpha$ (typically 0.05) before collecting data. This sets your threshold for statistical significance.

3

Collect Sample Data

Gather a representative sample and calculate relevant statistics (sample mean $\bar{x}$, sample standard deviation $s$, sample size $n$).

4

Calculate the Test Statistic

\text{Z-test: } z = \frac{\bar{x} - \mu_0}{\sigma/\sqrt{n}} \quad \text{or} \quad \text{T-test: } t = \frac{\bar{x} - \mu_0}{s/\sqrt{n}}
5

Determine the P-Value or Critical Value

Find the probability of observing a test statistic as extreme as yours (p-value), or compare against the critical value from the distribution table.

6

Make a Decision

If p-value < $\alpha$ (or test statistic falls in rejection region), reject $H_0$. Otherwise, fail to reject $H_0$.

7

State the Conclusion

Interpret your decision in the context of the original research question using non-technical language.
